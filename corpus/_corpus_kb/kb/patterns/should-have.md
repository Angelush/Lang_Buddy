## Form
should have + past participle

## Function
Se usa para hablar de una expectativa pasada que no se cumplió. A menudo implica que hubo un error o una oportunidad perdida.

## Examples
- "we should have given you room 402." — [[dialogues/en-b1-hotel-01]]
- "I should have checked the website before booking." — [[dialogues/en-b1-hotel-01]]

## Contrasts
Diferente de "should + base verb" (que se refiere al presente o futuro). Al añadir "have", movemos la estructura al pasado.
