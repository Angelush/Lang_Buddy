## Form
"a double-edged sword"

## Function
Se utiliza para describir algo que tiene consecuencias tanto positivas como negativas, donde los beneficios pueden verse compensados por las desventajas. Es una metáfora para indicar que una misma herramienta o situación puede beneficiar y perjudicar al mismo tiempo.

## Examples
- "It’s a double-edged sword, isn’t it? Useful tools that eventually start using us." — [[dialogues/en-b2-technology-01]]
- "Social media is a double-edged sword; it connects us with friends globally but can also increase anxiety." — [[dialogues/en-b2-technology-01]]

## Contrasts
En español se traduce literalmente como "arma de doble filo". Se usa de forma idéntica en contextos formales e informales.
