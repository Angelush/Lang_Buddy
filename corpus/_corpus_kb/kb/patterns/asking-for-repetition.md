## Form
Can you repeat [that/it]? / Sorry, can you repeat that?

## Function
Se utiliza cuando el hablante no ha escuchado o no ha comprendido lo que se acaba de decir y desea solicitar una repetición de forma educada, especialmente en un entorno escolar o formal.

## Examples
- "Sorry, can you repeat that?" — [[dialogues/en-a1-classroom-01]]
- "Can you repeat the instructions, please?" — [[dialogues/en-a1-classroom-01]]

## Contrasts
En español es muy común usar "¿Cómo?" para pedir que repitan, pero en inglés, "¿How?" puede sonar confuso o brusco. Es preferible usar una frase completa como "Can you repeat that?" para ser más educado y claro.
