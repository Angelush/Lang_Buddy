## Form
There + used to be + [sustantivo]

## Function
Se utiliza para describir un estado o una situación pasada que no es real en el presente. Equivale a "antes había" o "solía haber".

## Examples
- "**There used to be** a small cinema there, didn't there?" — [[dialogues/en-b1-hometown-01]]
- "**There used to be** a library here, but now it's a co-working space." — [[dialogues/en-b1-hometown-01]]

## Contrasts
A diferencia de "there was", "there used to be" enfatiza fuertemente el contraste con el presente: el objeto o situación ya no existe.

## Anticipation hint
"Before the shopping mall, ______ a beautiful park." (El estudiante debe predecir *there used to be*).
