## Form
"He/She is [adjective] / has [feature]"

## Function
Para describir el físico, usamos el verbo "to be" para la estatura o complexión (ej: *He is tall*) y el verbo "to have" para el pelo, ojos, etc. (ej: *She has long hair*).

## Examples
- "He looks quite tall." — [[dialogues/en-a2-describing-people-01]]
- "He has short, dark hair." — [[dialogues/en-a2-describing-people-01]]

## Contrasts
En español usamos el verbo "tener" para la altura (ej: "tiene 1.80m"), pero en inglés siempre usamos el verbo "to be" para describir la altura con adjetivos (ej: *He is tall*, no *He has tall*).
