## Form
**It is/was + [focus] + that/who/which + [clause]**
**What/All + [focus] + is/was + [clause]**

## Function
Se usa para destacar una parte concreta de la oración, muy útil en contextos profesionales donde es necesario clarificar o enfatizar un punto específico en una discusión o negociación.

## Examples
- "What concerns me is that such a shift fundamentally alters the logic we initially agreed upon." — [[dialogues/en-c1-negotiation-01]]
- "It was the social media API that triggered the need to redesign the architecture." — [[dialogues/en-c1-negotiation-01]]

## Contrasts
En español se usa una estructura similar con "Lo que... es..." (ej: *Lo que me preocupa es que...*), pero el inglés permite mayor flexibilidad con *it is... that* para enfatizar cualquier elemento de la oración (sujeto, objeto, adverbial).
