## Definition
Proceso metodológico en el que personas reales prueban una aplicación, sitio web o interfaz para validar su usabilidad, ergonomía y experiencia antes del lanzamiento definitivo.

## Examples
- "Grâce aux tests utilisateurs, nous avons identifié et corrigé une faille ergonomique majeure dans notre nouvelle application de streaming." — [[dialogues/fr-c1-negociation-01]]
- "Ah, mais c’est justement ces tests utilisateurs qui m’intéressent le plus !" — [[dialogues/fr-c1-negociation-01]]

## Notes
- Término muy utilizado en el ámbito del UX (User Experience) y desarrollo ágil.
- A menudo se considera una parte "no negociable" de un proyecto de calidad, aunque a veces se intenta eliminar para reducir costes.
