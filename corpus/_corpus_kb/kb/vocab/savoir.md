## Definition
En francés, el verbo **savoir** se utiliza para expresar el conocimiento de un hecho, una información o para indicar que se posee una habilidad aprendida (como tocar un instrumento, programar o realizar una tarea técnica). Es un verbo esencial para manejar información objetiva.

## Examples
- "Je sais, mais si nous ne respectons pas le calendrier, le client sera mécontent." — [[dialogues/fr-b2-travail-01]]
- "Savez-vous si les ajustements techniques sur l'application ont été validés par l'équipe ?" — [[dialogues/fr-b2-travail-01]]

## Notes
- **Savoir vs. Connaître:** Un error común es confundirlos. *Savoir* se utiliza con hechos, información ("savoir que...") o verbos en infinitivo ("savoir faire..."). *Connaître* se utiliza con personas, lugares o cosas concretas ("connaître quelqu'un", "connaître une ville").
- **Savoir + Infinitivo:** Cuando *savoir* va seguido de un verbo en infinitivo, indica una habilidad o técnica que se ha aprendido a realizar.
- **Registro:** Es un verbo de uso cotidiano y neutral, fundamental en un entorno formal de trabajo para confirmar la recepción de información o habilidades.
