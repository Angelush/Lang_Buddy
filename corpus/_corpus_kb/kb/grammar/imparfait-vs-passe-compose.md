## Rule
El *imparfait* se utiliza para describir estados, acciones en curso, descripciones o hábitos en el pasado (lo que hacíamos con frecuencia o el contexto de una situación). El *passé composé*, por otro lado, se utiliza para acciones puntuales, completadas o una secuencia de eventos definidos en el pasado (lo que ocurrió y terminó). Es la distinción fundamental para contar historias o eventos de forma coherente.

## Examples
- "J'y pensais déjà l'année dernière, mais j'ai décidé de me lancer maintenant." — [[dialogues/fr-b1-projets-avenir-01]]
- "Quand je travaillais dans ce café, je préparais souvent des gâteaux maison." — [[dialogues/fr-b1-projets-avenir-01]]

## Common errors
Los estudiantes suelen utilizar el *passé composé* para descripciones (por ejemplo: *"J'ai eu une idée" en lugar de "J'avais une idée"*) o el *imparfait* para acciones que son claramente puntuales y terminadas. También existe la tendencia a traducir directamente del español, donde la distinción entre pretérito indefinido y pretérito imperfecto no siempre se alinea perfectamente con el uso francés.

## When the agent should correct vs. let it slide
El tutor debe corregir este punto cuando el uso incorrecto del tiempo verbal altera la cronología o la naturaleza de la acción (si es hábito vs. puntual), ya que esto es esencial en el nivel B1 para mantener la precisión en la comunicación. Si el error es menor y no afecta la comprensión del contexto narrativo, se puede dejar pasar para fomentar la fluidez.
