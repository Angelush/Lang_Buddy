## Rule
Las oraciones de relativo sirven para añadir información sobre un sustantivo mencionado anteriormente. Usamos "who" para personas, y "which" o "that" para cosas o animales. Estas cláusulas definen o identifican al sustantivo de manera precisa.

## Examples
- "The show has too many characters who don't matter." — [[dialogues/en-b1-opinions-film-01]]
- "That is the streaming app which I use to watch movies." — [[dialogues/en-b1-opinions-film-01]]

## Common errors
Los hispanohablantes tienden a usar "that" para referirse a personas (donde es preferible "who") o a omitir el pronombre relativo cuando es necesario. También es común intentar usar "what" como relativo, lo cual es incorrecto en esta función.

## When the agent should correct vs. let it slide
Corregir si el error impide la comprensión, por ejemplo, usando "which" para personas. Si el estudiante usa "that" para personas, es aceptable en un registro informal, así que se puede dejar pasar.
