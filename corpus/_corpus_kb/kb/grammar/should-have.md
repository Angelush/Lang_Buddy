## Rule
Se utiliza "should have" + participio pasado para expresar arrepentimiento sobre algo que no ocurrió en el pasado, o para señalar una obligación o consejo que no se cumplió. Indica que algo hubiera sido mejor si se hubiera hecho de otra manera.

## Examples
- "He should have explained his plan earlier." — [[dialogues/en-b1-opinions-film-01]]
- "I should have watched the finale last night." — [[dialogues/en-b1-opinions-film-01]]

## Common errors
El error más frecuente es la confusión fonética o escrita de "should have" con "should of" (debido a la contracción "should've"). También es común olvidar el participio y usar el verbo en infinitivo (ej. "should have explain").

## When the agent should correct vs. let it slide
Corregir siempre "should of", ya que no es una forma válida, sino una confusión auditiva. Asegurarse de que el estudiante usa el participio correcto después de "should have".
