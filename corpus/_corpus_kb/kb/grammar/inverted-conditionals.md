## Rule
La inversión en oraciones condicionales es una estructura formal que omite la conjunción *if* e invierte el orden del sujeto y el verbo auxiliar.
1. **Had:** Se utiliza en el tercer condicional (*Had the system been programmed...*).
2. **Were:** Se utiliza en el segundo condicional (*Were we to ignore the pattern...*).
3. **Should:** Se utiliza en el primer condicional, a menudo con un sentido de menor probabilidad (*Should you find any errors...*).

## Examples
- "Had the system been programmed with true intent, it would have felt the grief." — [[dialogues/en-c2-ai-creativity-01]]
- "Were we to ignore the pattern recognition, we would be blind to the AI's limitations." — [[dialogues/en-c2-ai-creativity-01]]

## Common errors
El error habitual es olvidar la inversión una vez omitido el *if* (ej. *"If I had known"* -> *"Had I known"* es correcto, pero *"Had I known"* no debe confundirse con *"If I had known"* mal estructurado como *"Had I have known"*).

## When the agent should correct vs. let it slide
Al ser una estructura altamente formal y característica de un nivel C2, el agente debe corregir activamente cualquier error de sintaxis en la inversión. Si el estudiante usa condicionales estándar en un contexto donde se esperaría una inversión (por formalidad), el agente puede sugerir la inversión como una mejora estilística, pero no es estrictamente un error.
