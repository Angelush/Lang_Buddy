## Rule
"Used to" se emplea para describir hábitos, acciones o estados que ocurrían en el pasado de forma repetida, pero que ya no suceden en la actualidad. Se usa con el infinitivo del verbo. Estructura: *Used to + verb*.

## Examples
- "I used to like sci-fi, but this show is too confusing." — [[dialogues/en-b1-opinions-film-01]]
- "They used to watch shows on television, but now they only stream." — [[dialogues/en-b1-opinions-film-01]]

## Common errors
Los hispanohablantes suelen decir "I use to" en lugar de "I used to" (confundiendo la forma pasada con el infinitivo). También tienden a sobreusar "used to" para acciones puntuales del pasado en lugar de usar el pasado simple.

## When the agent should correct vs. let it slide
Corregir siempre la forma "I use to" cuando el contexto es pasado, ya que es un error gramatical claro. Es importante distinguir entre un hábito pasado ("used to") y una acción única.
